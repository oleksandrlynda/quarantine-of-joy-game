02 / Scene application

Same arena. Better hierarchy.

The upgraded pass does not need more total objects. It needs fewer repeated primitives, stronger clusters, open shooting lanes, and a small number of memorable vertical anchors.

03 / Placement recipe

Variation comes from composition, not randomness.

A useful kit can still produce a noisy scene. Lock these ratios into the arena generator or use them as the manual dressing checklist.

The 60 / 30 / 10 rule

Build calm first. Accent last.

Control both color and object density. The player should immediately know what is playable, what is important, and what is dangerous.

60%quiet base

Art-direction decision
The world should feel like a quarantine assembled yesterday on top of a place that existed for decades.
Keep

Barrels, direct geometry, chunky enemy shapes, broad color fields, fast silhouettes.

Change

Anonymous walls, repeated cubes, full-field grass, equal object spacing, unbroken horizon bands.

Add

Gates, mesh, circles, wedges, cables, towers, damage states, route markers, and civilian traces.
